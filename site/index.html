perintah mkdocs

  • mkdocs new [nama-dir] - Membuat project baru.
  • mkdocs serve - Memnajalankan serve lokal sambil mengubah dokumen.
  • mkdocs build - Build atau mengubah menjadi html.
  • mkdocs help - Menampilkan perintah bantuan di mkdocs.
